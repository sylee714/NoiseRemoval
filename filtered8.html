jsoup: Java HTML Parser

jsoup is a Java library for working with real-world HTML. It provides a very convenient API for fetching URLs and extracting and manipulating data, using the best of HTML5 DOM methods and CSS selectors.

jsoup implements the specification, and parses HTML to the same DOM as modern browsers do.

  • scrape and HTML from a URL, file, or string
  • and extract data, using DOM traversal or CSS selectors
  • the HTML elements, attributes, and text
  • user-submitted content against a safe white-list, to prevent XSS attacks
  • tidy HTML

jsoup is designed to deal with all varieties of HTML found in the wild; from pristine and validating, to invalid tag-soup; jsoup will create a sensible parse tree.

Example

Fetch the homepage, parse it to a DOM, and select the headlines from the In the news section into a list of (, ): 


Document doc = Jsoup.connect("https://en.wikipedia.org/").get();
log(doc.title());
Elements newsHeadlines = doc.select("#mp-itn b a");
for (Element headline : newsHeadlines) {
 log("%s\n\t%s", headline.attr("title"), headline.absUrl("href"));
}
